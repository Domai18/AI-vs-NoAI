# AI-vs-NoAI

# No “Happy” AI: A Critical Reflection

## Overview
This project is a written analysis examining the limitations of framing artificial intelligence as inherently neutral or beneficial. It explores how optimism-driven narratives can obscure ethical, social, and political consequences of AI deployment.

## Focus
The essay considers:
- Human bias embedded in algorithmic systems
- The gap between technical performance and social impact
- The responsibility of technologists beyond model accuracy

## Context
This work reflects an interest in AI ethics and governance, developed alongside technical training in data analytics and computer science.

## Format
The project is presented as a written analytical piece rather than code, emphasizing clarity of argument and critical reasoning.

## Note
This work is independent and does not reflect proprietary research or institutional affiliations.
